# 👨‍💼 Employee Management System

A robust Java-based Employee Management System designed to efficiently manage employee records, attendance, performance points, and bonus calculations. This project demonstrates the practical implementation of Object-Oriented Programming (OOP), CRUD operations, and Java Collections Framework concepts in a real-world business scenario.

---

## 🚀 Features

✅ Add New Employees

✅ View Employee Records

✅ Search Employees by ID

✅ Update Employee Information

✅ Delete Employee Records

✅ Track Employee Attendance

✅ Manage Performance Points

✅ Calculate Employee Bonuses

✅ Menu-Driven Console Interface

✅ Object-Oriented Design

---

## 📋 System Functionalities

### Employee Management
- Add employee details
- Update employee information
- Delete employee records
- Search employees using unique IDs

### Attendance Tracking
- Monitor employee attendance
- Maintain attendance records

### Performance Evaluation
- Assign performance points
- Track employee achievements

### Bonus Calculation
- Automatic bonus calculation based on:
  - Attendance
  - Performance Points

---

## 🎯 Bonus Logic

| Attendance | Performance Points | Bonus |
|------------|-------------------|--------|
| 25+ Days | 100+ Points | 10% Salary Bonus |
| Any | 50+ Points | 5% Salary Bonus |
